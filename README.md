# GraphX

Graphical interface for interacting with a graph database.

## Getting Started

### Installation

Install the client dependencies
```
$ cd ./graphx-client && npm install
```
### Running in dev mode

Launch the app in development mode
```
$ cargo tauri dev
```

### Creating a build
Create the build
```
$ cargo tauri dev
```
